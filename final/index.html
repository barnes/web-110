Final Project

Your final project is extremely open-ended. The goal is to demonstrate a deep understanding of the basics of HTML & CSS by building one final and complete project. How you build, and what your build is completely up to you. You'll have the bulk of the last 2 classes to work on this.

Use any previous code or resources from class, or any tutorial, guide, etc. Feel free to use Chatbots to help develop content and ideas, but avoid having the chatbot generate HTML & CSS, especially if it generates code you do not understand.

Requirements

  • Multiple pages, at least 3
  • Clean, readable content that is easy to navigate
  • Robust CSS that styles the text styles and overall layout
  • Minimal, modern styling, the site should look professional and modern
  • Semantic HTML used throughout, code should be readable and clean
  • Use of flexbox / CSS Grid for multi-column layouts
  • Plenty of images, which can be sourced from services like Unsplash
  • Deployed and available online, either via Github Pages, Glitch, Codepen, etc.

Deliverables

  1. Your published site URL
  2. Your code itself, either a public repo, or the files themselves.

Bonus Challenges

  • Responsive design that adapts to screen sizes
  • Javascript for user interaction
  • CSS Animations
  • Use a minimal CSS framework like PicoCSS

Advice

  • Focus on the layout and styling first, content second. Get the bulk of the site 'scaffolded' first.
  • Incrementally add additional things like CSS libraries, JS, etc.
  • Use chatbots to help guide your design and content.
  • Use CSS snippets from sites like Codepen to have unique elements
  • Create psuedo-templates, re-usable code snippets you can copy and paste throughout your site. You can stash these in a different directory.
  • Don't be afraid to throw things away and start over
